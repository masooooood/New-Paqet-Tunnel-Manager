کپی شده از پروژه ای که پایین نوشتم و تغییراتی دادم . تانل من پکت دیگه نیست پاکت شده 

مدیریت پیشرفته تونل و فوروارد پورت



🇬🇧 English

A powerful and interactive Bash-based tunnel manager for advanced port forwarding and tunnel management.

Designed for simplicity, speed, and automation.


## 🚀 Quick Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/masooooood/New-Paqet-Tunnel-Manager/main/paqet-manager-range.sh)
```

---




✨ Features

🔹 Support for single ports and port ranges (e.g. 60000-60070)

🔹 Apply one protocol selection to all selected ports

🔹 TCP / UDP / TCP+UDP support

🔹 Automatic iptables configuration

🔹 Clean interactive CLI interface

🔹 Lightweight and fast

🔹 No external dependencies

📦 Traffic Modes
Mode	Description
Port Forwarding	Forward specific ports or ranges
SOCKS5 Proxy	Create a SOCKS5 proxy tunnel
🔐 Protocol Options

1 → TCP

2 → UDP

3 → TCP + UDP

🇮🇷 فارسی

یک اسکریپت قدرتمند و تعاملی مبتنی بر Bash برای مدیریت تونل و فوروارد پورت به صورت حرفه‌ای.

طراحی شده برای سادگی، سرعت و اتوماسیون کامل.

✨ امکانات

🔹 پشتیبانی از پورت تکی و رنج پورت (مثلاً 60000-60070)

🔹 اعمال یک انتخاب پروتکل برای کل رنج پورت

🔹 پشتیبانی از TCP / UDP / هر دو

🔹 تنظیم خودکار iptables

🔹 محیط کاربری تمیز و مرحله‌ای

🔹 سبک و سریع

🔹 بدون وابستگی خارجی

📦 حالت‌های ترافیک
حالت	توضیح
Port Forwarding	فوروارد پورت‌های مشخص یا رنج
SOCKS5 Proxy	ایجاد تونل SOCKS5
🔐 انتخاب پروتکل

1 → فقط TCP

2 → فقط UDP

3 → TCP و UDP همزمان

🛠 پیش‌نیازها

سرور Ubuntu / Debian

دسترسی Root

نصب بودن iptables
